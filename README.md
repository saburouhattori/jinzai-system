# 人材管理システム (Jinzai System)

## 概要 (Overview)

Google Apps Script (GAS) と Google スプレッドシートを活用した人材紹介事業向けの管理システムです。
候補者（登録者）の管理、求人案件の管理、面接から採用（内定）までのプロセスを一元管理し、業務の効率化を図ります。日本およびミャンマー拠点のスタッフが共同で運用することを想定して設計されています。

## 主な機能 (Features)

* **候補者管理**:
  「登録者マスタ」への新規登録、更新、削除。顔写真データを含んでいても高速に一括更新可能なロジックを実装。

* **年齢自動計算**:
  スプレッドシートの DATEDIF 関数と ARRAYFORMULA を活用し、生年月日から満年齢をリアルタイムに自動計算。メンテナンスフリーな運用を実現。

* **案件・採用管理**:
  求人案件の登録、修正、削除機能。および案件に対する候補者の「採用（内定）」登録機能。

* **面接履歴の自動追記**:
  「採用者登録」実行時に、該当案件で面接を受けた全候補者（採用・不採用問わず）の「面接履歴」と結果を、「登録者マスタ」へ自動で改行追記します。

* **自動リスト同期 (動的マッピング)**:
  登録者マスタの更新内容を「採用者一覧」「未採用者一覧」「簡易リスト」へ自動同期。ヘッダー名（1行目の文字列）をキーにしてデータを自動振り分けするため、列の追加や移動にも柔軟に追従します。

* **直感的なUI / 待機表示**:
  データの送受信や同期など、時間のかかる非同期処理中には画面全体を覆うローディングスピナーを表示し、システムの状態を視覚的にフィードバックします。

* **ファイルリンク連携**:
  関連ファイルURLをGoogleドライブのファイル名を取得して「スマートチップ風」に自動変換。

* **履歴書・簡易リスト生成**:
  候補者IDを指定して、提出用の履歴書や簡易リストを自動生成する機能。

* **支払い管理連携（外部連携）**:
  外部シート「海外ミャンマー管理リスト【Funtoco】」へ採用者情報をボタン一つで同期。重複を検知し、外部で入力された金額等のデータは保護しつつ基本情報のみを更新可能。

## システム構成 (System Structure)

### スプレッドシート（データベース）

* **登録者マスタ**: 全候補者のベースデータ（ID: SD-XXXX）

* **案件管理**: 求人案件データ（ID: JOB-XXXX）

* **リスト・出力用**: 未採用者一覧, 採用者一覧, 簡易リスト, 履歴書, 面接管理

* **各種マスタ**: 事業者マスタ, 送り出し機関マスタ, 日本語学校マスタ, 企業情報, 面談用質問

### Google Apps Script (バックエンド・フロントエンド)

* 01\_共通・メニュー.gs / JS_Common.html : 共通処理・UIの制御・メニュー構築

* 02\_候補者管理.gs / HTML_Candidate.html / JS_Candidate.html : 候補者データ操作

* 03\_案件・採用管理.gs / HTML_Job.html / JS_Job.html : 案件データ操作・採用登録

* 04\_事業者管理.gs / HTML_Company.html : 事業者データ操作

* 05\_マスタ連携・その他.gs : リストの自動同期（syncListSheets）、外部シート連携、辞書生成などの補助機能

* 履歴書作成.gs : 履歴書の自動出力処理

* MainSidebar.html / Sidebar_CSS.html : サイドバーのUIとスタイリング
